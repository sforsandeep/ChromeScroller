ChromeScroller

Privacy Policy

Last updated: March 2026

No data collected · No servers · 100% local

Overview

ChromeScroller is a browser extension that lets you inspect, highlight, and screenshot any DOM element on a web page — including full-height scroll-stitch captures of scrollable containers. It operates entirely within your local browser and does not collect, store, or transmit any data to any server or third party.

Data we do NOT collect

ChromeScroller does not collect, access, store, or share any of the following:

  • Personally identifiable information (name, email, address, etc.)
  • Browsing history or visited URLs
  • Page content, text, or HTML from any website you visit
  • Screenshots or captured images — these are saved directly to your device only
  • Mouse position, click data, or any user activity logs
  • Authentication credentials, cookies, or session tokens
  • Device information, IP address, or location

How the extension works

When you activate ChromeScroller, it injects a content script into the active tab. This script listens for mouse movement and scroll-wheel input to let you navigate and select elements on the page. When you click an element:

  • The browser takes a screenshot of the visible tab area using Chrome's built-in captureVisibleTab API.
  • If the element is scrollable, the extension scrolls through it and stitches multiple strips into one full-height image — entirely in memory, in your browser.
  • The final PNG is saved to your local Downloads folder using Chrome's downloads API.

No screenshot data, no page content, and no user information is ever sent outside your device. The extension has no backend, no cloud storage, and no analytics.

Permissions used

  • activeTab — to inspect the page DOM and measure element dimensions
  • scripting — to inject the capture tool when you activate the extension
  • downloads — to save the screenshot PNG to your Downloads folder
  • tabs — to call captureVisibleTab() for taking the browser screenshot

Third-party services

ChromeScroller does not use any third-party analytics, crash reporting, advertising, or data-processing services. No data is shared with any third party under any circumstances.
